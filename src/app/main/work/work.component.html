Etnica Eco-Friendly

Sitio web con galería de fotos, carta diaria, chat para consultas, pedidos y reservaciones con envío automático de correos.

Implementado en Firebase:

  • Hosting para el despliegue.
  • Storage para imágenes.
  • Database para la carta.
  • Functions para notificaciones por e-mail.
  • Authentication: control de acceso para la administración del sitio.

Interfaz de gestión desarrollada con Angular y Bootstrap.

  • Stack: Angular, Firebase, GitHub
  • URL: https://etnica-restaurant.web.app
